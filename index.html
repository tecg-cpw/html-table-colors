Les nuanciers en informatique En informatique aussi ont été développés des nuanciers. Contrairement aux « grands » nuanciers classiques, le but ici n'est pas de recenser un grand nombre de couleurs, mais principalement de simplifier la façon de les désigner, en les nommant. Les noms HTML Le langage HTML donne à la base un nom à 16 couleurs. Lorsque vous concevez vos pages pour un site Internet, vous pouvez utiliser ce nom HTML au lieu de leur code RVB.
